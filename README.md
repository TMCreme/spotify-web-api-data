## Running the code 



### Sample config.ini
```
[CLIENT_CREDENTIALS]
CLIENT_ID=
CLIENT_SECRET=
AUTH_URL=
BASE_URL=

[AUTHCODE_FLOW]
BASE_URL=https://accounts.spotify.com/authorize?
RESPONSE_TYPE=code
SCOPE=user-library-read
REDIRECT_URI=http://localhost:8000/
```
